4プロジェクト体制 — 理想像（5/27確定）

5/27のビジョン整理MTGで、これからの会社は4つのプロジェクトが並走する形に整理された。 各プロジェクトに「理想像 / 現状 / ギャップ」を当てる。

① LDM（Light Diamond Method）

理想像サブスク3階層完成 — 無料1300人 / 8,800円層 / 上位2-4万層。トップ講師1人が2,000万安定化 → 2-3人に拡張 → 将来的にFC化。
現状(5/27)アースジャーニーオプチャ 1300人達成。8,800円サブスク 67名。上位コースは未展開。
主要ギャップ上位2-4万層の設計・展開がこれから。トップ講師1人へのフォーカス化と2,000万安定化の道筋づくり。
次の一手7月: 上位コース設計開始 / トップ講師フォーカス本格化。

② シンクロ起業塾

理想像常時 40-100人 + 3ヶ月毎フォーラム継続。自分基準の豊かさ循環を体現する成功事例が2-3人 → 5-10人に増える。
現状(5/27)約40人前後、成功事例 2-3人。
主要ギャップリーダー層へのアプローチ（自己犠牲層からの脱却）。教育設計の言語化（学び方 / 自責思考 / 根底教育 / 日本人OS）。
次の一手3ヶ月毎フォーラムの設計を固定化。成功事例のストーリー収集と再現性ある仕組み化。

③ OTO

理想像純度レベル0-4のコミュニティが回り始める。新音源を継続リリース。
現状(5/27)新音源完成。純度レベル定義済み。コミュニティ運用はこれから。
主要ギャップ純度レベル0-4の運用設計（参加体験・進階導線）。新音源リリースの継続リズム。
次の一手純度レベル運用ルールの仕組み化。リリースペースのスケジュール化。

④ 社外プロジェクト

理想像神田さん新企画始動（6月〜）。リンポチェ様事務局サポート立ち上がり。
現状(5/27)神田さんオファー受領済み。リンポチェ様事務局サポートはこれから。
主要ギャップ立ち上げそのもの。堀内さんが消耗しない関与設計（仕組み・事務局体制）が必要。
次の一手6月: 神田さん新プロジェクト始動 / リンポチェ様事務局オペレーション設計開始。
4プロジェクトを横串で見る判断軸： 純度（どれだけ本質に近いか）/ 共鳴度（参加者が自然と動き出すか）/ 仕組み化（堀内さん個人に依存せず回るか）。
どのプロジェクトも、この3軸でメンテナンスし続ける。
