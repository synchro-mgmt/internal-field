MOMENT

メイク・美しさ・意識の変化を通じて、鏡を見た瞬間に「これが私だ」と自分の存在へ戻り、内的安心感と外に出る力を同時に取り戻すプロジェクト。

COREプロジェクトの核

メイクで別人になる場所ではない。内側に閉じる美しさでも、外側に奪われる美しさでもない。鏡を見た瞬間に、「これが私だ」と自分の存在に戻り、外の世界にも出ていける場所。

MOMENTの中心は、美容そのものではなく、外見を通じて自己認識・内的安心感・健全な他者評価・行動変化が一気に立ち上がる瞬間にある。

2026-06-30 UPDATEさえさえさんMTGで深まった論点

新しい核

内的安心感と外的評価の統合

内面だけ整えても安心領域に閉じやすく、外見だけ磨いても評価依存に寄りやすい。MOMENTは「私が私である安心感」と「外に出て評価を受け取れる力」の両方を扱う。

7月検証

ライブ配信でコンセプトを育てる

完成した定義を一方的に出すのではなく、体験者の声・参加者の反応・内田さんの美の定義を拾いながら、MOMENTの外向き言語を磨く。

入口設計

オプチャは無料リストではなく文化の場にする

コメントや感想が歓迎される空気を作り、ここだけで有料級の価値がある場として設計する。菱田さん側のコミュニティメンバーも運営候補に入れる。

素材化

鏡を見た瞬間をダイジェスト映像にする

ビフォー、施術中、鏡を見た瞬間、アフター、感想を短い映像にまとめ、ライブ配信・オプチャ誘導・LPに使える資産にする。

POSITIONING会社設計上の位置づけ

会社設計

意識領域を外側から見える形に翻訳する

堀内さんの意識・自己実現領域を、美容・鏡・ビフォーアフターという可視化しやすい体験に接続する。

協業

内田裕士さん側の美容・ブランド文脈と接続する

美塾、ブランド開発、社会的信用のある美容文脈と、MOMENTの深い変化を無理なくつなぐ。

体験価値

菱田紗絵子さんの変化を言語化する

人からどう見られるかへの恐れがほどけ、自分の望みを伝え、気後れしていた場所に行けるようになる変化を素材化する。体験者/紹介者として場を立ち上げる立ち位置も設計する。

信用設計

スピリチュアルに見えすぎない見せ方を作る

「波動」ではなく「意識」「内的安心感」「自己実現」「美しさ」「ビフォーアフター」を中心に対外言語を設計する。

CHANGEMOMENTで起きている変化

変化領域起きていること事業上の意味
自己認識鏡を見た瞬間に「これが私だ」と感じ、自分そのものへの肯定が起きる。単なる美容ではなく、体験価値として差別化できる。
内的安心感外側に安心感を埋めてもらおうとしなくなり、自分単体で存在できる感覚が立つ。自己実現や次の行動へ進む前段階になる。
行動変化自分の望みを伝え、気後れしていた場所に行けるようになる。ビフォーアフターと体験談にしやすい。
他者評価どう見られるかへの恐れが減り、健全な評価は受け取れるようになる。社会や場に出る力として説明できる。

ROADMAP内部ロードマップ

PHASE 0 / 立ち上げ

GitHub管理の開始

private repo、README、初回MTG議事録、プロジェクト設計、コンセプトメモ、ロードマップ、初期Issuesを作成済み。

PHASE 1 / コンセプト確定

「内側と外側の統合」を一文定義にする

6/30 MTGで見えた「内的安心感だけでも、外見/他者評価だけでもない」という核を、LP・オプチャ・ライブ配信用の言葉にする。

PHASE 2 / 7月ライブ配信

コンセプトを育てる公開対話を実施する

7月中に最低1回、可能なら2回。堀内さん・内田さん・菱田さんの立ち位置を整理し、体験者の声やビフォーアフターを使って外向き言語を検証する。

PHASE 3 / オプチャ設計

入口と文化づくりの場を作る

オプチャ名、初回投稿、1週間分投稿テーマ、コメント歓迎の空気、返信担当、上位体験への導線を設計する。

PHASE 4 / ダイジェスト映像

鏡を見た瞬間を伝わる素材にする

ビフォー、施術中、鏡を見た瞬間、アフター、感想をつないだ短尺映像を検討する。制作担当、素材使用許可、4-5万円程度の予算可否を確認する。

PHASE 5 / 次回MTG

内田さんと実施条件を決める

内田さんの美の定義、ライブ配信可否、素材使用可否、次回MOMENT実施への温度感、7-9月の役割分担を確認する。

PHASE 6 / company-brain接続

会社設計の正式プロジェクトへ接続する

company-brain側へ置く情報とMOMENT repo側に置く情報の境界を決め、必要なら反映案を作る。
